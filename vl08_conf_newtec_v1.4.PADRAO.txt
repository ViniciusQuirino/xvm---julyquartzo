	- Buzzer alterado para OUT1
	
- Versão 1.3
	- Adicionado cadastro de redes Wi-Fi
	- Reporte de conexão via Wi-Fi alterado para RUV00

- Versão 1.4
	- Alterado evento ED058 de keep alive para enviar via GR0 ou WF0
	- Alterado TD de keep alive do TD01 para o TD06 para não conflitar com TD de tracking
	- Alterado reporte de keep alive para GP para reduzir consumo
	- Ajustes no salvamento de velocidade e RPM máximos
